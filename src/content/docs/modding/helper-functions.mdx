As discussed in [Lua: Abridged for XDRV](/xdrv-charting-guide/modding/lua), helper functions can make programming mods in XDRV a lot easier and much more succinct. If you have one or two functions that are generally useful for a lot of modfiles, you may find yourself copying and pasting the functions from file to file. If you don't have any functions like that yet, maybe you'll be able to find one here!

This page contains several helper functions by various XDRV charters. All helper functions in the repository can be viewed as separate files on this page. If you would prefer to read these helper functions from the GitHub itself, you can do so by going to the [code directory](https://github.com/theblockiest/xdrv-charting-guide/tree/master/src/assets/code) of the GitHub repository. I recommend looking at the `all.lua` file in each subdirectory, which combines all functions and function documentation from the individual files in that folder.

### Pulse Functions

***Author:* TheBlockiest**
<Badge text="Events" variant="note" /><Badge text="Alpha" variant="note" />

These functions create a "pulse" effect in the background, increasing the background's alpha value to one value, holding it for some (or no) time, and then decreasing it to another value. Pulses can be great for representing percussive elements like kicks, snares, and crashes. These pulse function use linear easing for alpha only.

import pulse from '../../../assets/code/pulse/pulse.lua?raw'
import pulseLeft from '../../../assets/code/pulse/pulseLeft.lua?raw'
import pulseRight from '../../../assets/code/pulse/pulseRight.lua?raw'
import pulseBoth from '../../../assets/code/pulse/pulseBoth.lua?raw'

### Split Alpha Functions

***Author:* TheBlockiest**
<Badge text="Events" variant="note" /><Badge text="Alpha" variant="note" />

import splitAlpha from '../../../assets/code/split-alpha/splitAlpha.lua?raw'
import mergeAlpha from '../../../assets/code/split-alpha/mergeAlpha.lua?raw'

These functions swap the values of `PathAlpha` with the values of `LeftPathAlpha` and `RightPathAlpha`. This allows you to seamlessly swap whether alpha values are controlled together or individually.

***Author:* TheBlockiest**
